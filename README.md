# IT News / AI Assistant Bot

Ko‘p funksiyali Telegram bot:
- AI helper va oddiy suhbat
- translate, summary, plan
- code explain, code review, bug helper
- roadmap, interview, challenge
- GitHub repo info va trending
- voice/audio -> text transcript
- Uzbek IT news auto-post
- quiz va leaderboard

## Asosiy buyruqlar
- `/ask savol`
- `/translate matn`
- `/summary matn`
- `/plan maqsad`
- `/daily`
- `/bug xato`
- `/explain kod`
- `/review kod`
- `/project mavzu`
- `/challenge javascript`
- `/roadmap backend`
- `/interview backend`
- `/news uzbek`
- `/transcribe ielts` keyin voice yuborish

## Deploy
Render Web Service:
- Build: `npm install`
- Start: `node src/app.js`
- `RENDER_EXTERNAL_URL` ni Render URL ga teng qiling.
